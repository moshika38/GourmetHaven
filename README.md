# Gourmet Haven

**Gourmet Haven** is a modern, responsive restaurant website built with **Next.js**.  
It showcases the restaurantâ€™s menu, location, contact info, and other key details, with full-stack functionality including payments and authentication.

## Features

* Built with **Next.js**  
* Responsive layout (mobile + desktop)  
* Dynamic menu pages  
* Contact form section  
* **Stripe** payment integration  
* **Firebase Authentication** (Google / Email login)  
* **MongoDB** database for storing users, orders, and menu items  
* Easy to deploy
